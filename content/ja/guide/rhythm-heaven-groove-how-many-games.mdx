<Section icon={Target} title="ソロリズムゲーム：80種類以上のミニゲームコレクション">

本作の核となるのは、ソロ用のリズムゲームです。一人で遊べる**ゲーム数**が気になる方も多いかと思いますが、その答えは80種類以上のユニークなシチュエーションが用意されています。これらのゲームは、画面上の視覚的な仕掛けや予期せぬテンポチェンジに対処しながら、一定のビートを刻む能力が試される、短時間でエネルギッシュなチャレンジです。

画面上を流れるノーツ（音符）に合わせる一般的なリズムゲームとは異なり、『リズム天国 ミラクルスターズ』では音声ガイド（合図）とキャラクターのアニメーションが重要になります。例えば「フープ転がし（Hoop Trundling）」での「パ・ピ・プ・ペ・ポ」のような特定の音を聴き取り、正確なタイミングでボタンを押す必要があります。テーマは日常の家事から、シュールなSFバトルまで多岐にわたります。

| ゲーム名 | タイプ | 主なルール/操作 |
| :--- | :--- | :--- |
| **フープ転がし (Hoop Trundling)** | ソロ | 音声ガイドの最後の拍に合わせてフープを飛び越える |
| **ポップ・ステップ・ロール (Hop Stop N Roll)** | ソロ | テンポに合わせて転がったりジャンプしたりする猫の人形を操作する |
| **フルーツフレックス (Fruit Flex)** | ソロ | 力こぶを作って、飛んでくるフルーツをタイミングよく弾き飛ばす |
| **スライス＆ダイス (Slice N Dice)** | ソロ | 目まぐるしいキッチンで、飛んでくる野菜を刻む |
| **シンクロウィング (Synchro Wings)** | ソロ | 鳥の群れとシンクロして羽ばたく |
| **リズム脱毛 (Rhythm Tweezers)** | ソロ / マルチ | リズムに乗って動くタマネギから毛を抜き取る |
| **ソーダホップ (Soda Hop)** | ソロ | 炭酸ソーダボトルのリズムに合わせてジャンプする |

<Callout type="warning" title="重要">
画面の見た目だけに頼らないようにしましょう。80種類以上のゲームの中には、視覚的な引っ掛けや、画面が真っ暗になる「ブラインド」演出が含まれるものが多数あります。音楽と音声ガイドをしっかり聴いて、耳でリズムを刻むことが攻略の鍵となります。
</Callout>

</Section>

<Section icon={Swords} title="マルチプレイヤーモードとパーティプレイ">

みんなでワイワイ遊びたいプレイヤーにとって、マルチプレイヤー要素は大きな見どころです。『リズム天国 ミラクルスターズ』には、**30種類以上のローカルマルチプレイヤーゲーム**が収録されています。これらは1台の本体で1〜4人プレイができるよう特別に設計されており、長時間のゲームプレイよりも、短時間で笑えるチャレンジを求める「ゲームナイト」やパーティーに最適です。

マルチプレイヤーゲームは、主に2つのカテゴリに分かれています：
1. **協力チャレンジ**: プレイヤー全員で協力して同じリズムを維持します。誰か1人でも合図を逃すと、チーム全体のスコアに影響することがあります。
2. **対戦バトル**: 誰が最も正確なタイミングで入力できるかを競います。「ジャスト」の判定にどれだけ近かったかに応じてポイントが加算されます。

<CardGrid cols={2}>
  <Card title="協力：テニスクエスト (Tennis Quest)">
    - **共通の目標**: リズムに乗ってモンスターを倒す。
    - **ルール**: プレイヤーが交代でテニスボールを打ち返す。
    - **勝利条件**: チーム全体の合計精度（正確さ）に依存する。
  </Card>
  <Card title="対戦：ケーキウェイト (Cake Wait)">
    - **共通の目標**: 最も大きなケーキのカットを手に入れる。
    - **ルール**: 時計がぴったり「3秒」を指すのを待つ。
    - **勝利条件**: 最も早く正確に反応できたプレイヤーが勝利する。
  </Card>
</CardGrid>

<Callout type="info" title="マルチプレイの要件">
各プレイヤーにそれぞれ対応するコントローラーが必要です。Joy-Conでプレイ可能ですが、4人フルで遊ぶ場合は追加のコントローラーが必要になる場合があります。マルチプレイは「TVモード」と「テーブルモード」に対応していますが、「携帯モード」はシングルプレイヤー専用となります。
</Callout>

</Section>

<Section icon={Sparkles} title="ビートスペル：リズムRPGモード">

「ビートスペル（Beatspell）」は、本作における最も注目すべき新要素です。これはアンロック式のシングルプレイヤーモードで、従来のリズムアクションがRPGスタイルのバトルシステムへと進化しています。単にメダル獲得を目指してステージをクリアするだけでなく、タイミングよくボタンを押すことで魔法を唱え、キャラクターを回復し、ストーリーに沿ってモンスターを撃破していきます。

メイン的80種類以上のソロゲームを進めてメダルを集めることで、ビートスペルのストーリーの新しいチャプター（章）が解放されていきます。このモードは、これまでのシリーズ作品には少なかった「長期的な成長と進行」の楽しさを提供してくれます。

| 要素 | ビートスペルのシステム |
| :--- | :--- |
| **戦闘** | モンスターのビートに合わせてタップし、炎や氷の魔法を唱える |
| **回復** | 戦闘中に特定のリズムパターンを入力してHPを回復する |
| **キャラクターの成長** | ハイスコアを獲得してキャラクターのステータスをレベルアップさせる |
| **アンロック要素** | ソロゲームで獲得したメダル数に応じて新しいチャプターが解放される |
| **エネミー（敵）** | ボス戦では、複雑で複数のテンポが入り混じるリズムパターンが登場する |

<Callout type="success" title="アンロックのコツ">
まずは「フープ転がし（Hoop Trundling）」や「フルーツフレックス（Fruit Flex）」などの序盤のソロゲームで、最低でも「ハイレベル（Superb）」の評価を狙いましょう。これにより、プレイ開始後かなり早い段階で「ビートスペル」の最初の3チャプターを解放するのに必要なメダルが集まります。
</Callout>

</Section>

<Section icon={Shield} title="体験版（スターターデモ）と始め方">

ゲームの難易度が心配な方は、無料の**『リズム天国 ミラクルスターズ 体験版（スターターデモ）』**から始めてみるのが最適です。この体験版では、製品版のコンテンツから厳選された一部をプレイできます。シリーズの基本である「聴く、合図、アクション」のサイクルを学ぶために作られています。

<Steps>
  <Step num="1" title="体験版をダウンロードする">
    Nintendo Switchのニンテンドーeショップにアクセスし、「リズム天国 ミラクルスターズ」（欧州地域では「Rhythm Paradise Groove」）と検索して、無料の体験版（スターターデモ）をダウンロードします。
  </Step>
  <Step num="2" title="ソロゲームをマスターする">
    収録されている「フープ転がし（Hoop Trundling）」を含む5つのソロゲームをプレイしましょう。これらのゲームを通じて、ビートを維持することや音声ガイドに反応する基本を学べます。
  </Step>
  <Step num="3" title="ローカルマルチプレイを試す">
    友達を誘って「リズム脱毛（Rhythm Tweezers）」に挑戦してみましょう。これは体験版に収録されている唯一のマルチプレイヤーゲームで、協力プレイの素晴らしい入門編となります。
  </Step>
  <Step num="4" title="セーブデータを引き継ぐ">
    製品版の購入を決めた場合、体験版のセーブデータを引き継ぐことができます。獲得したメダルや進行状況はすべて製品版にキャリーオーバーされます。
  </Step>
</Steps>

<Callout type="info" title="体験版の収録内容">
体験版には、**5種類のソロ用リズムゲーム**と**1種類のマルチプレイヤーゲーム**が収録されています。「ビートスペル」モードや、製品版にある詳細なキャリブレーション設定は含まれていません。
</Callout>
