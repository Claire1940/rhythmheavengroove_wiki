<Section icon={Sparkles} title="ナイトモードの仕組みを理解する">

2026年7月2日のゲームリリースにより、プレイヤーには新たなリズムの挑戦が提示されました。中でも最も過酷な追加要素が、この高難易度の別キャンペーンモードです。このモードでは、標準ステージのビジュアルがネオンに照らされた夜の設定へと変化します。しかし、変化は単なる見た目だけではありません。

このモードでは、各ミニゲームのテンポが速くなり、視覚的な合図は影、点滅するネオン、あるいは雨や霧といった天候エフェクトによって頻繁に遮られます。この設計により、プレイヤーはほぼ完全に聴覚的な記憶とリズムの同期に頼ることを余儀なくされます。「パーフェクト」や「ハイレベル」評価を得るための判定枠は大幅に縮小されており、極めて精密なボタン操作が要求されます。

| 特徴 | デイモード（通常） | ナイトモード |
| :--- | :--- | :--- |
| **視覚的合図** | 明確で明るく、追いやすい | 影、ネオンの点滅、天候により遮られる |
| **テンポ倍率** | 1.0x（標準速度） | ステージにより 1.15x から 1.35x |
| **判定枠** | 余裕あり（ハイレベルで約50ms） | タイト（ハイレベルで約25ms） |
| **オーディオトラック** | オリジナルの楽器構成 | 高速テンポ、シンセビート、裏打ちのシンコペーション |
| **ミス許容度** | 3回程度の大きなミスが可能 | 即失敗、または厳しい評価ダウンを招きやすい |

<Callout type="tip" title="オーディオへの依存">
これらのステージでは目に頼ってはいけません。視覚的な妨害は、意図的にタイミングを狂わせるために配置されています。必要であれば目を閉じ、ドラムのパターンや裏打ちのシンセの合図だけに集中してください。

<Section icon={Target} title="ナイトモードの解放方法">

これらの高速リズムチャレンジに挑戦するには、メインキャンペーンを進める必要があります。ゲーム開始直後からアクセスすることはできません。以下の手順に従ってモードを解放してください。

<Steps>
  <Step num="1" title="メインキャンペーンをクリアする">
    標準モードの全80種類のシングルプレイヤーゲームをクリアしてください。「フープころがし」から最終リミックスステージまで、すべてのステージで少なくとも「平凡」以上の評価を得る必要があります。
  </Step>
  <Step num="2" title="ビートスペルのモンスターを倒す">
    シングルプレイヤー用の「ビートスペル」モードを進めてください。リズムに基づいた魔法を使い、ボスを撃破します。このモードのクリアは、クリア後のショップ解禁の必須条件です。
  </Step>
  <Step num="3" title="フローボールを蓄積する">
    ワールドマップにランダムに出現する「パーフェクトキャンペーン」を成功させ、少なくとも15個のフローボールを獲得してください。
  </Step>
  <Step num="4" title="ナイトチケットを購入する">
    ゲーム内ショップを訪れ、集めたフローボールを使ってアイテム「ナイトチケット」を購入してください。これにより、メインメニューに別のゲートが恒久的に解放されます。
  </Step>
</Steps>

<Callout type="warning" title="ビートスペルの要件">
ビートスペルモードでは、攻撃魔法や防御魔法を放つために正確なタイミングが求められます。後半のボス戦では連続してリズムを外すと体力が一気に削られるため、回復魔法の練習も忘れずに行ってください。
</Callout>

<Section icon={Swords} title="高度なタイミングとシステムのキャリブレーション">

判定枠が非常に狭いため、システムの遅延（レイテンシ）がプレイに致命的な影響を与える可能性があります。最新のディスプレイでのプレイやワイヤレスオーディオ機器の使用は、数ミリ秒の遅延を引き起こし、「パーフェクト」評価の獲得を不可能にする場合があります。

| プラットフォームモード | 推奨オーディオ遅延設定 | 最適なコントローラー設定 | 接続タイプ |
| :--- | :--- | :--- | :--- |
| **TVモード** | -4 ～ -6 フレーム | 有線コントローラー | HDMI直結（レシーバーなし） |
| **テーブルモード** | 0 フレーム | Joy-Con取り付け | 直接接続 |
| **携帯モード** | 0 フレーム | Joy-Con取り付け | 直接接続 |
| **ワイヤレスオーディオ** | -8 ～ -12 フレーム | 有線コントローラー | Bluetooth（非推奨） |

| 評価 | 判定枠 (ms) | スコア倍率 | 視覚インジケーター |
| :--- | :--- | :--- | :--- |
| **パーフェクト** | 15ms 未満 | 3.0x | 黄金のフラッシュ |
| **ハイレベル** | 16ms - 30ms | 2.0x | 輝く星のバースト |
| **平凡** | 31ms - 60ms | 1.0x | シンプルな火花 |
| **やりなおし** | 60ms 超 | 0.0x | 赤いバツ印 |

<Callout type="info" title="キャリブレーションの確認">
セッションを開始する前に、必ずオプションメニューにある内蔵キャリブレーションツールを使用してください。メトロノームに合わせて10回タップすることで、ゲームが平均的な入力遅延を計算し、自動的にオーディオ同期を調整します。
</Callout>
