<Section icon={Swords} title="リズム天国 ミラクルスターズ ビートスペルモード">

シリーズへの大きな追加要素は、専用のシングルプレイヤーリズムRPGである **「ビートスペルモード」** です [1]。単発のステージをクリアする代わりに、プレイヤーはリズム入力が呪文、防御、剣の一撃に直結するファンタジーの世界へ旅立ちます [1]。

### ビートスペルRPGの仕組み

| RPG要素 | リズム天国での対応アクション | 機能と進行 |
| :--- | :--- | :--- |
| **呪文詠唱** | リズミカルなボタンコンボ [1] | 特定のビートパターンを入力して火、氷、雷を放つ [1]。 |
| **回復** | 裏打ちのリカバリー [1] | ゆっくりとした防御リズムに合わせてタップしHPを回復 [1]。 |
| **モンスターバトル** | ボス戦 [1] | 敵の攻撃リズムに合わせてパリィやカウンターを行う [1]。 |
| **進行状況** | メダル収集 [1] | 通常のソロゲームでメダルを獲得し、新しい章を解放する [1]。 |

```
通常のソロプレイ ---> メダル獲得 ---> ビートスペルの章を解放 ---> モンスターとバトル
```

<Callout type="info" title="ビートスペルの解放方法">
ビートスペルはゲーム開始直後からは遊べません。メインのソロリズムゲームの初期セットを進め、最初のシルバーメダルセットを獲得することで、このRPGアドベンチャーへのポータルが解放されます [1]。
</Callout>

</Section>

<Section icon={Shield} title="マルチプレイとキャリブレーション設定">

リズムゲームには絶対的な精度が必要です。現代の薄型テレビは表示遅延の度合いが異なるため、「ハイレベル」や「パーフェクト」の評価を得るためには、正しく設定することが不可欠です [1]。

### ローカルマルチプレイの設定（1〜4人用）
マルチプレイモードでは、1台のNintendo Switch本体で最大4人まで遊べます [1]。
- **コントローラー要件**: 各プレイヤーに1つの対応コントローラー（Joy-Con横持ちやProコントローラーなど）が必要です [1]。
- **推奨モード**: TVモードとテーブルモードが強く推奨されます [1]。携帯モードでのマルチプレイも可能ですが、画面が狭く感じられる場合があります [1]。
- **オンライン非対応**: ローカル通信およびオンラインマルチプレイには対応していません。すべてのプレイは1つの画面でのローカルプレイとなります [1]。
