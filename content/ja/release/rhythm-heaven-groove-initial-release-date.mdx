<Section icon={Shield} title="操作方法、アクセシビリティ、設定">

2026年のリリースにおいて、アクセシビリティは大きな焦点となっています。本作は、すべてのプレイヤーが自分のリズムを見つけられるよう、複数の操作体系をサポートしています。物理ボタンの触覚フィードバックを好む方も、音のみのプレイを必要とする方も、設定メニューで細かく調整可能です。

特筆すべき機能は、画面上のテキストを読み上げるゲーム内アシスタント「**リール・ミス・リード（Li'l Miss Reeds）**」です。これは特に年少のプレイヤーや視覚障害のある方に役立ちます。さらに、現代のテレビディスプレイでの入力遅延を解消するための包括的なキャリブレーションツールも含まれており、ハイレベルなプレイには欠かせません。

| 設定項目 | 機能 | メリット |
| :--- | :--- | :--- |
| **キャリブレーション** | 手動オフセット | テレビの音と映像のズレを修正 |
| **オーディオ合図** | 低音強化 | ビートを感じやすくする |
| **テキスト読み上げ** | リール・ミス・リード | 音声ガイド付きのメニューと合図 |
| **入力モード** | ボタンのみ | シリーズ伝統の操作スタイル |
| **視覚補助** | 高コントラスト | 合図の視認性を向上 |
