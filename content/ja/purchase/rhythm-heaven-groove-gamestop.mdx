<Section icon={Target} title="本体セットアップ＆TVキャリブレーションガイド">

『リズム天国 ミラクルスターズ』は視覚的な反応スピードではなく、フレーム単位の正確な入力に依存するため、入力遅延はゲーム体験を大きく損なう原因になります。一般的な現代のテレビは、映像処理の過程でわずかな音・映像のズレを発生させます。これに対処するため、開発チームはボタンを押すタイミングを音楽と正確に同期させる「ゲーム内キャリブレーション設定」を用意しています。

リズムゲームを最高のパフォーマンスで楽しむために、以下の手順でハードウェアを最適化してください：

<Steps>
  <Step num="1" title="TVのゲームモードを有効にする">
    テレビの画質設定メニューを開き、内部の映像処理遅延を最小限に抑えるために「ゲームモード」を有効にします。
  </Step>
  <Step num="2" title="キャリブレーションメニューを開く">
    Nintendo Switchでゲームを起動し、「設定 > サウンド＆キャリブレーション」に移動します。
  </Step>
  <Step num="3" title="音のビートに合わせる">
    画面のインジケーターを見ずに、目を閉じて、スネアドラムの合図が聞こえた瞬間に正確にボタンをタップします。
  </Step>
  <Step num="4" title="ミリ秒単位のオフセット値を保存する">
    調整された遅延スコア（ミリ秒単位で表示）を確認し、設定を保存してからミニゲームを開始します。
  </Step>
</Steps>

プレイ環境によって、必要なキャリブレーション設定は異なります。以下の調整マトリクスを参考に、最適な基準値を見つけてください：

| プレイ環境 | 推奨キャリブレーション設定値 | オーディオ出力の推奨度 |
| :--- | :--- | :--- |
| **携帯モード** | 0 ms 〜 +5 ms | 高 (本体のイヤホンジャック直挿し) |
| **テーブルモード** | 0 ms 〜 +10 ms | 高 (本体スピーカーまたは有線ヘッドホン) |
| **TVモード (ゲームモードON)** | +10 ms 〜 +25 ms | 中 (HDMI経由でTVスピーカーから出力) |
| **TVモード (通常画質モード)** | +40 ms 〜 +80 ms | 低 (TVの通常映像処理による遅延を回避推奨) |
| **ワイヤレスBluetoothオーディオ** | +100 ms 〜 +200 ms | 極めて低 (リズムゲームでの使用は非推奨) |

<Callout type="tip" title="有線オーディオの使用を推奨">
Switchでプレイする際は、Bluetoothヘッドホンの使用を避けてください。Bluetoothは大きなオーディオ遅延を引き起こし、ゲーム内のキャリブレーション機能でも完全に補正しきれない場合があります。最高の体験を得るには、本体スピーカーまたは有線ヘッドホンを使用してください。
</Callout>

</Section>

<Section icon={Trophy} title="ビートスペルモード＆ローカルマルチプレイのセットアップ">

『リズム天国 ミラクルスターズ』には、これまでのシリーズ作とは一線を画す2つの大きな追加要素があります。それが、RPG風の**「ビートスペルモード」**と、拡張されたローカルマルチプレイ機能です。「ビートスペル」は、ボタンを押すタイミングが魔法の呪文、回復魔法、戦闘アクションに変換され、モンスターを倒していくアンロック式のシングルプレイヤーアドベンチャーです。通常スコアを競ういつものモードとは一味違う、新鮮な体験を楽しめます。

また、パーティーや集まりに最適なマルチプレイ機能は、1台の本体で最大4人まで同時に楽しむことができます。オンラインマルチプレイには対応していないため、すべての対戦・協力プレイは同じ画面でのローカルプレイとなります。

| プレイヤー人数 | 対応コントローラー | 推奨ゲームモード |
| :--- | :--- | :--- |
| **1人** | Joy-Con / Proコントローラー | ソロミニゲーム、ビートスペルモード |
| **2人** | Joy-Con 2本 (おすそわけ) | リズムおはし、テニスアドベンチャー協力プレイ |
| **3〜4人** | Joy-ConまたはProコンの組み合わせ | ケーキ運び、生き残りバトルシナリオ |

<Callout type="info" title="マルチプレイのボットサポート">
4人のプレイヤーが集まらない場合でも、空いている枠にはコンピューターが操作するボット（CPU）が自動的に入ります。これにより、2人しかプレイヤーがいない場合でも、30種類以上のすべてのマルチプレイシナリオを体験することができます。
</Callout>
