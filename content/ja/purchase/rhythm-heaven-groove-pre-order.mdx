ひとり用ゲームは、それぞれ求められるリズムの操作性によって分類されています。シンプルなタップだけで遊べるものから、複雑なパターンや「裏拍」での操作が求められるものまで様々です。

| ゲーム名 | タイプ | 主な特徴・操作 |
| :--- | :--- | :--- |
| **輪くぐり** | ひとり用 | 最後の「ポ」の合図に合わせてジャンプ |
| **フルーツフレックス** | ひとり用 | 筋肉をアピールしてリズムよくフルーツを弾く |
| **スライス＆ダイス** | ひとり用 | 飛んでくる野菜を順番にタイミングよく刻む |
| **リズム脱毛** | マルチプレイ | 協力してタイミングよく毛を抜く |
| **テニスクエスト** | マルチプレイ | テニスのスイングでモンスターを撃退 |
| **ケーキウェイト** | マルチプレイ | 「3」のカウントに合わせてタイミングよく掴む |

<Callout type="info" title="ビートスペルの解放条件">
「ビートスペル」は最初からプレイできるわけではありません。ひとり用リズムゲームの最初の3つのステージグループをクリアし、メダルを10枚以上集めることで、最初のRPGストーリーチャプターが解放されます。
</Callout>

<Section icon={Shield} title="操作方法、アクセシビリティ、および遅延調整">

**『リズム天国 ミラクルスターズ』の予約**がカジュアルゲーマーの間で非常に人気となった理由の一つに、操作が極めてシンプルであることが挙げられます。ほとんどのゲームは1つか2つのボタンしか使用しません。難しさは入力の複雑さではなく、リズムそのものにあります。

**アクセシビリティ機能:**
- **リル・ミス・リーズ**: 画面上のテキストや合図を音声で読み上げてくれるゲーム内アシスタント。
- **視覚的サポート**: 聴覚に障がいのある方向けに、ハイコントラストな視覚インジケーターのオン／オフ切り替えが可能。
- **遅延調整ツール**: 現代のテレビ環境で発生しやすい、映像と音声のわずかなズレ（レイテンシー）を調整可能。

| 設定項目 | 機能 | 推奨設定 |
| :--- | :--- | :--- |
| **テレビの調整** | ボタン入力と音声を同期 | サウンドバーを使用している場合は最初に実行 |
| **アシスタント音声** | メニューテキストの読み上げ | 小さなお子様がプレイする際におすすめ |
| **ビジュアルガイド** | 拍子に合わせて画面が点滅 | 周囲が騒がしい環境でプレイする際におすすめ |
| **入力モード** | ボタン vs タッチ（携帯モード） | 基本的にはボタン操作の方が正確でおすすめ |

<Callout type="warning" title="入力遅延に関する注意">
テレビモードでプレイする場合は、テレビの画質設定を「ゲームモード」に設定してください。通常の画像処理が行われると、最大100msの遅延が発生することがあり、評価「ハイレベル」や「パーフェクト」の獲得が非常に困難になります。
</Callout>

</Section>
