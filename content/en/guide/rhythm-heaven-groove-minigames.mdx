<Section icon={Shield} title="Calibration and Accessibility Settings">

Because timing is everything in this game, even the slightest display lag can throw off your performance. This is especially true when playing on modern TVs, which often introduce input latency. Fortunately, the developers have included robust calibration tools to ensure your button presses align perfectly with the audio cues.

### How to Calibrate Your Game
If you notice that your inputs are consistently marked as "late" or "early," navigate to the settings menu from the title screen. Run the TV input delay calibration test. You will be asked to tap the button along to a simple metronome beat with your eyes closed, allowing the system to automatically adjust the game's internal timing window.

### Accessibility Features
To make the game accessible to as many players as possible, Nintendo has introduced Lil' Miss Reeds, an in-game assistant character. Lil' Miss Reeds can read all on-screen text aloud, providing spoken guidance for menus, tutorials, and timing results. If you prefer a quieter experience, this text-to-speech feature can be easily toggled off in the settings menu.

<Callout type="warning" title="TV Game Mode">
Always ensure your television is set to 'Game Mode' before calibrating. This reduces hardware-level processing lag, giving you the most accurate window possible for challenging rhythm inputs.
</Callout>
