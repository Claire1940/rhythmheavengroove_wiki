<Section icon={Swords} title="조작 방법 및 협응 메커니즘">

단일 버튼을 사용하는 일반적인 리듬 게임과 달리, 크랩 스낵은 두 개의 서로 다른 입력을 동시에 조율해야 합니다. 게의 왼쪽과 오른쪽 집게발을 모두 조작하므로, 성공적으로 음식을 튕겨내려면 정확히 같은 순간에 두 버튼을 모두 눌러야 합니다.

| 액션 | 조작 입력 | 게임플레이 역할 |
| --- | --- | --- |
| **오른쪽 집게발** | Ⓐ 버튼 | 오른쪽 집게발을 올려 날아오는 음식을 튕겨냅니다. |
| **왼쪽 집게발** | ✚ / ◀ 방향키(D-Pad) 버튼 | 왼쪽 집게발을 올려 날아오는 음식을 튕겨냅니다. |
| **양손 동기화** | Ⓐ + ◀ 동시 입력 | 간식을 깔끔하게 받아 모래 구멍에 넣는 데 필요합니다. |

<Callout type="warning" title="양손 협응 경고">
왼쪽 집게발이 오른쪽 집게발과 싱크가 맞지 않으면 간식이 빠져나갑니다. 반드시 Ⓐ 버튼과 D-Pad의 왼쪽 버튼을 정확히 같은 프레임에 누르세요.
</Callout>

</Section>

<Steps>
  <Step num="1" title="집게발 감각 익히기">
    음악이 시작되기 전에 Ⓐ 버튼과 D-Pad의 ◀ 버튼을 다섯 번 눌러 양손의 감각을 조율하고 입력이 동시에 등록되는지 확인합니다.
  </Step>
  <Step num="2" title="음식 비주얼 확인하기">
    화면 오른쪽을 주시하세요. 마카롱이 굴러오는지 망고스틴이 굴러오는지 확인하여 타이밍 전략을 결정합니다.
  </Step>
  <Step num="3" title="마음속으로 박자 세기">
    일정한 배경 템포에 맞춰 발을 구르세요. 마카롱의 경우 다음 정박에 누릅니다. 망고스틴의 경우 엇박자(정박 사이의 반 박자 쉬는 타이밍)를 기다렸다가 누릅니다.
  </Step>
  <Step num="4" title="튕겨서 모래에 넣기">
    간식이 게의 위치에 도달하는 순간 두 버튼을 동시에 꾹 누릅니다. 성공하면 깔끔한 "퍼펙트!" 또는 "에이스!" 효과음이 들립니다.
  </Step>
</Steps>

### 타이밍 판정 기준
| 입력 타이밍 | 화면 표시 | 게임플레이 결과 |

| **퍼펙트 / 프레임 일치** | 퍼펙트! / 에이스! | 게가 음식을 깔끔하게 튕겨내어 모래 구멍에 넣습니다. |
| **약간 어긋남** | 빠름! / 느림! | 음식을 옆으로 비껴 튕겨내며, 동료 게들이 째려봅니다. |
| **입력 놓침** | 놓침... | 음식을 앞에 떨어뜨리며, 동료 게들이 화를 내며 째려봅니다. |
