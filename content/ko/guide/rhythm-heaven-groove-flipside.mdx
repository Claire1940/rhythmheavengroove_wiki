### 비트스펠 전투 방식

일반 아케이드 모드와 달리, 비트스펠 모드는 구조화된 전투 시스템을 제공합니다:

- **리듬 입력**: 일련의 비트가 표시된 주문 스크롤이 주어집니다. 시퀀스를 정확하게 탭하면 파이어볼이나 라이트닝 같은 주문을 시전합니다.
- **적의 턴**: 몬스터들은 자신만의 리듬 패턴으로 공격해 옵니다. 타이밍에 맞춰 실드 버튼을 정확히 눌러 피해를 막아야 합니다.
- **보스전**: 보스는 전투 중에 템포를 바꾸거나 무음 메커니즘을 사용하여 플레이어가 전적으로 내부 메트로놈에만 의존하도록 만듭니다.

### 필수 오디오 보정 단계

최신 TV에서 플레이하는 경우, 화면 처리 과정에서 시청각 지연이 발생할 수 있습니다. 다음 단계에 따라 게임을 보정해 보세요:

1. 메인 화면에서 **설정(Settings)** 메뉴로 이동합니다.
2. **오디오/비디오 보정(Audio/Video Calibration)**을 선택합니다.
3. 화면의 메트로놈 클릭 소리에 맞춰 버튼을 탭합니다.
4. 시스템이 디스플레이 지연 시간을 계산하여 입력 판정 범위를 자동으로 조정합니다.
5. 설정을 저장합니다. 휴대 모드에서 TV 모드로 전환하는 경우 이 과정을 반복하는 것이 좋습니다.

### 게임 편의 기능 옵션

닌텐도는 모든 플레이어가 음악을 즐길 수 있도록 다음과 같은 유용한 편의 기능을 도입했습니다:

- **릴 미스 리드(Li'l Miss Reeds) 음성 지원**: 게임 외 메뉴와 화면의 안내 텍스트를 읽어주는 옵션 음성 지원 기능입니다.
- **시각적 리듬 표시기**: 오디오 신호보다 시각적 피드백을 선호하는 플레이어를 위해 화면에 깜빡이는 프롬프트 표시를 켤 수 있습니다.
- **원버튼 컨트롤 모드**: 모든 조작을 단 하나의 버튼 입력으로 매핑하여 복잡한 미니게임을 단순화하고, 버튼 레이아웃 대신 순수하게 템포에만 집중할 수 있게 합니다.
